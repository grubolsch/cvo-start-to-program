Strings bij elkaar plakken (string concatenation)

Heel vaak ga je strings met elkaar moeten combineren, dit noemt "concatenation".

Je kan dit op verschillende manieren doen, wij gaan nu de + operator gebruiken.

Je hebt de plus al gebruikt om getallen op te tellen, maar je kan dus ook teksten bij elkaar plakken!

        const str = 'Hello' + ' ' + 'World';
        str; // 'Hello World'
        Je kan ook += gebruiken, dan is a += b een korte versie van a = a + b.

        // tweede voorbeeld
        let str = 'Hello';
        str += ' ';
        str += 'World';
        str; // 'Hello World'

Er zijn ook andere manieren om hetzelfde resultaat te krijgen, zoals met String literals, maar daar ga we niet naar kijken in deze module.
